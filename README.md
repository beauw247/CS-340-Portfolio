# CS 340 Portfolio

## About This Repository

This repository contains my CS 340 portfolio artifacts, including my final dashboard code, CRUD Python module, and Project Two README document.

## Portfolio Reflection

This project helped me understand why readable and organized code matters so much. In Project One, I created a CRUD Python module to handle the database connection and the basic create, read, update, and delete operations. That helped a lot in Project Two because I was able to reuse that module instead of stuffing all of the database logic directly into the dashboard code. It made the project easier to manage, easier to understand, and easier to change when something needed to be updated. I could see myself using a CRUD module like this again in the future for another dashboard, database app, or any project where I need to connect user features to stored data.

When I approach a problem as a computer scientist, I try to slow down first and figure out what the client actually needs. For the Grazioso Salvare dashboard, I had to look at the rescue dog requirements and then break the project into smaller parts, like the filters, data table, chart, and map. This project felt different from some earlier assignments because it was not just about writing code to get an answer. It was about making something useful that a person could actually interact with. In the future, I would use this same kind of approach by starting with the client’s goals, figuring out what data matters most, building the queries carefully, and testing each part as I go.

Computer scientists build tools that help people use information in better ways. That matters because data by itself is not always useful until it is organized and displayed clearly. For a company like Grazioso Salvare, this dashboard could help staff find dogs that match specific rescue training needs without having to search through records manually. The filters, table, chart, and map make the data easier to work with and easier to understand. This type of project showed me how computer science can help a company save time, reduce mistakes, and make better decisions.
